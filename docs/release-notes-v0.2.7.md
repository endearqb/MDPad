# MDPad v0.2.7 更新说明

发布日期：2026-04-23

## 亮点更新
- HTML 预览进入更可用的编辑阶段：
  - 普通 HTML 文档现在支持更稳定的预览内编辑工作流，阅读与演示模式也进一步收口，适合在页面视图里直接校对和调整内容。
  - HTML 预览顶部工具栏已简化，不再长期暴露“预览 / 编辑”切换，常用模式切换和页码/文档控制更聚焦。
- 图表编辑体验完成一轮集中升级：
  - 图表数据弹窗现在更接近轻量表格编辑器，标签与系列可直接在数据视图中编辑、移动和删除。
  - 运行时图表预览更稳妥，无法复用脚本时会回退到静态快照，避免出现“能打开但没有可用预览”的情况。
  - 图表结构菜单已改为独立浮层，靠近表格边缘时也会自动贴边或翻转，不再被滚动容器裁剪。
- 阅读与展示体验继续收口：
  - HTML 预览的滚动条现已与主编辑界面风格统一，宽表格、页面视图和预览 iframe 的细节更一致。
  - 图表数据表头和输入单元格宽度、对齐和交互留白已修正，整体观感更稳定。

## 体验优化
- 图表数据编辑弹窗的标签/系列菜单已改为更紧凑的图标化操作入口，同时保留提示文案和无障碍名称。
- 表头、角单元格、数值单元格的布局已统一，横向滚动场景下更接近规则网格，不容易再出现错位。
- HTML 预览中的通用可视编辑入口已从主工具栏收口，交互层级更清晰，减少模式切换干扰。

## 技术调整
- 为图表结构菜单新增了独立的定位 helper 与回归测试，覆盖右侧贴边、底部翻转和窄视口 clamp 等边界场景。
- 为 HTML 预览、图表数据编辑和相关 UI 状态切换补充了测试，降低后续迭代回归风险。
- 构建侧继续收敛了预览与编辑相关实现边界，便于后续在 HTML 预览、图表编辑和 slide 模式上继续演进。
