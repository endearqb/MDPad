# MDPad v0.2.5 更新说明

发布日期：2026-04-18

## 亮点更新
- 外部文件变更处理更稳妥了：
  - 新增 `手动确认` / `自动更新` 两种模式，便于按你的工作习惯选择同步方式。
  - 当文件已被其他应用修改时，MDPad 会先提示再处理，避免你在不知情的情况下被静默覆盖。
  - 保存前会额外检查磁盘版本；如果外部文件已经变化，应用会阻止直接覆盖，降低误覆盖风险。
- 页面刷新后的恢复体验更完整了：
  - 右键刷新或 WebView 重载后，当前文档与草稿状态可以继续恢复，不会轻易回到空白页。
- Markdown 编辑器稳定性提升：
  - 修复了文档重载、切换或整篇内容替换时，选区可能越界并导致编辑器异常的问题。

## 体验优化
- 窗口最小尺寸现在会被原生层和运行时双重约束，避免窗口被缩到过小后出现布局异常。
- “打开示例文档”已修复安装版路径错误，安装后可直接打开内置示例。
- 标题栏与状态栏字重、滚动条细节做了收敛，整体观感更接近轻量原生编辑器。
- 外部变更相关文案改得更直白，优先使用“手动确认”“自动更新”“立即更新”这类非技术表达。

## 技术调整
- 清理了 `src-tauri/src/lib.rs` 中已被导出服务替代的旧实现，减少冗余代码与构建 warning。
- 为窗口尺寸、刷新恢复、外部文件检测等链路补充了回归测试，提升后续迭代的可维护性。
