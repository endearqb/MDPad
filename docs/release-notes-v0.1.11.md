# MDPad v0.1.11 更新说明

发布日期：2026-03-05

## 亮点修复
- 修复多窗口场景下的焦点层级问题：当已打开 3 个文档窗口后再打开第 4 个及后续窗口，新窗口现在会正确置顶并获得焦点。

## 技术调整
- 后端窗口创建链路在构建成功后新增显式激活步骤：
  - `show()`
  - `unminimize()`
  - `set_focus()`
- 保留原有失败回滚逻辑，窗口创建失败时仍会清理对应初始状态，避免脏状态残留。
