# 富文本混合粘贴崩溃二次加固

## Summary
- 在表格混合粘贴修复基础上，新增统一富文本粘贴准备入口。
- 表格 HTML 仍优先走专用清洗、受控 `pasteHTML` 和 `fixTables` 兜底。
- 非表格但明显来自网页/Office/Google/复杂 wrapper 的 HTML 现在会先清洗，再受控粘贴；`pasteHTML` 异常会降级到纯文本 fallback。

## Changes
- `richHtmlTables.ts` 新增 `prepareRichHtmlPaste` 和 `applyPreparedRichHtmlPaste`。
- 清洗范围扩展到 `figure/figcaption`、`details/summary`、`dl/dt/dd`、完整 HTML 文档片段、Office/Google 复制噪声、SVG/MathML 噪声和危险属性。
- `MarkdownEditor` 的早期 paste 分支改为统一处理富文本 payload，保留现有媒体、Markdown 和只读拦截行为。
- `richHtmlTables.test.ts` 增加非表格混合富文本、fallback 抛错和常见光标上下文回归。

## Verification
- `pnpm exec tsc --noEmit`
- `pnpm exec vitest run src/features/editor/clipboard/richHtmlTables.test.ts`
- `pnpm test -- clipboard`
- `pnpm test -- markdownCodec`
- `pnpm test`
- `pnpm build`

## Known Boundaries
- 通用路径只接管外部/复杂 HTML，不拦截简单内部 HTML 或纯文本 Markdown。
- 未引入 Tauri clipboard 插件，也未引入付费 TipTap paste handler。
- 复杂 HTML 若清洗后仍无法被 TipTap schema 接收，会降级为纯文本以保证不崩溃。

# Markdown 表格单元格最小宽度调整

## Summary
- 将 Markdown 表格单元格最小宽度从 `120px` 调整为 `40px`。
- 保留表格整体 `min-width: 100%`，单列表格仍会填满编辑器内容区。

## Changes
- `MD_TABLE_CELL_MIN_WIDTH` 改为 `40`。
- 现有 `--md-table-cell-min-width` CSS 变量和 TipTap 表格 resize 配置继续复用同一个常量。

## Verification
- `pnpm exec vitest run src/features/editor/markdownCodec.test.ts src/features/editor/clipboard/richHtmlTables.test.ts`
- `pnpm build`
