# 修复窗口按钮、HTML 预览全屏与 16:9 预览比例

## Summary
- 关闭按钮 hover 现在与最小化、主题切换按钮一致，不再使用红色 hover 背景。
- HTML 预览 iframe 获焦后会把 `F11` / `Escape` 快捷键上报给父层 App，避免全屏后点进预览页面无法退出。
- 应用全屏时，鼠标移到窗口顶部会显示右上角退出全屏按钮，点击即可退出。
- `16:9 Slide` 改为 HTML 预览内部 16:9 舞台开关，不再调整整个应用窗口尺寸。

## Implementation
- `htmlPreviewDocument` 新增 `mdpad:html-preview:fullscreen-shortcut` 消息，iframe 内 capture 监听 `F11` / `Escape` 并通过既有 token/source 校验链路传回父层。
- `HtmlPreview` 新增全屏状态回调和 `isSlideAspect` 布局开关；开启时 iframe 位于居中的 16:9 stage 内，关闭后恢复填满预览区域。
- `TopBar` 的 `16:9 Slide` 菜单项改为调用 App 内部 toggle；旧的 16:9 窗口尺寸 helper 和对应测试已移除，避免再次误用为窗口级预设。
- App 根层新增全屏退出按钮，并补充中英文 `Exit fullscreen` / `退出全屏` 文案。
