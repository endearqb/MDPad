# v0.2.10 发布记录

## Summary
- 本次发布聚合了 HTML/SVG 预览编辑稳定性、窗口比例模式调整，以及富文本混合粘贴崩溃修复。
- 当前发布版本为 `0.2.10`，安装包为 `MDPad_0.2.10_x64-setup.exe`。
- GitHub Release 使用 `docs/release-notes-v0.2.10.md` 作为发布说明。

## Paste Fix
- 编辑器主链路已直接接入 `tiptap-markdown@0.8.10`，兼容当前 Tiptap 2.27.2。
- 配置与验证项目保持一致：`Markdown.configure({ html: true, transformCopiedText: true, transformPastedText: true })`。
- HTML 富文本粘贴不再由 MDPad 手动 `pasteHTML` 或 sanitizer 抢先接管，交回 Tiptap/ProseMirror 原生粘贴路径；Markdown 文本粘贴/复制由 `tiptap-markdown` 处理。
- 粘贴错误 toast 已改为短文案并支持长文本换行/滚动，避免 ProseMirror fragment dump 挤出 toast。

## Paste 待完善
- 继续观察 `tiptap-markdown` 对本项目自定义节点的覆盖：数学公式、callout、Obsidian 图片 embed、媒体块和复杂表格若出现序列化差异，再为对应扩展补充 markdown storage spec。
- `markdownCodec.ts` 暂时保留给 PDF/export/diagnostics 等非编辑器主链路；后续可逐步收敛到统一 serializer。
- sanitizer 模块保留为专项修复工具，但不再挂在普通 HTML 富文本粘贴主路径。
