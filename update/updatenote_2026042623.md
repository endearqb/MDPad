# v0.2.10 发布记录

## Summary
- 本次发布聚合了 HTML/SVG 预览编辑稳定性、窗口比例模式调整，以及富文本混合粘贴崩溃修复。
- 当前发布版本为 `0.2.10`，安装包为 `MDPad_0.2.10_x64-setup.exe`。
- GitHub Release 使用 `docs/release-notes-v0.2.10.md` 作为发布说明。

## Paste Fix
- 富文本粘贴现在在 ProseMirror 默认解析前增加受控入口。
- “标题/正文 + 表格”的网页复制内容会优先走表格清洗路径，规范空单元格、inline-only 单元格、外部 wrapper 和表格噪声。
- Word/Google/网页复制出的非表格复杂 HTML 会走轻量 sanitizer；`pasteHTML` 失败时降级到纯文本 fallback，避免进入 `MDPad startup failed`。

## Paste 待完善
- Tiptap Paste Rules 适合 Markdown-like 文本模式，例如 `**bold**`、图片 Markdown、URL 或固定文本触发节点，不适合作为复杂 HTML 富文本粘贴的主修复方案。
- Word、Google Docs/Sheets、Office/Outlook、Notion/Confluence、AI 聊天网页等来源仍应继续沉淀 source-specific normalizer fixture。
- 后续可以把当前 sanitizer 进一步拆成独立 HTML sanitizer 模块，并评估通过 Tiptap `transformPastedHTML` 覆盖更多非崩溃但格式脏的默认粘贴路径。
- 合并单元格、嵌套表格和源站自定义 HTML 目前以“不崩溃、尽量保留结构、失败降级文本”为优先级，后续再做更细的格式还原。
