# 更新说明（2026-04-22 23:00）

## 本轮更新
- HTML 预览顶部工具栏已收口，不再显示“预览”“编辑”；当前保留“阅读 / 演示 / 自动 / 幻灯片 / 文档 / 页码”，并支持再次点击“阅读”或“演示”返回普通预览。
- 通用 HTML 可视编辑链路已下线：旧的 `visual-edit` 子模式、元素检查器、元素选中与 patch 提交入口已移除，但双击文字编辑和图表编辑弹窗仍保留。
- HTML 预览 iframe 现在会继承宿主主题的滚动条 token，主窗口滚动条和 HTML 内部表格/容器滚动条已统一到和 Markdown 预览一致的样式。
- 图表数据编辑进一步收口为“结构菜单 + 数值网格”的混合模式：标签和系列通过局部菜单完成编辑、移动、删除，表格主体继续专注于数值录入。
