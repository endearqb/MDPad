# 更新说明（2026-04-25 10:00，SVG 编辑修复）

## 本轮更新
- 恢复 SVG 编辑链路：iframe 内 SVG 右键/双击会把编辑请求发送到父层 `HtmlPreview`，父层打开 SVG 编辑窗并在应用后写回 HTML 状态。
- 修复 SVG 保存回流：`applySvgPatch` 成功后会调用 `commitVisualHtmlChange` 并刷新 iframe 预览，关闭 SVG 编辑窗后页面中的 SVG 立即显示新内容，同时进入未保存状态。
- 放大连接线选择热区：`line`、`polyline`、`path` 默认使用不少于 18px 的命中范围，折线和正交 path 优先按线段距离计算，bbox 只作为无法解析时的兜底。
- 恢复 SVG 编辑器几何 helper 与测试，继续保留当前图表编辑器和 runtime chart 逻辑。
- 调整 SVG 文字编辑视觉：文字内容编辑只显示 textarea 输入框，去掉外层 field/card 视觉，输入框圆角跟随 `--editor-surface-radius`。
