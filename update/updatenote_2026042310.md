# 更新说明（2026-04-23 10:00）

## 本轮更新
- 图表数据编辑弹窗中的标签头、系列头菜单已从纵向下拉样式收口为横向工具条式浮层，视觉方向与 markdown 划词菜单保持一致。
- 标签头和系列头 trigger 内的三个点图标已移除；现在直接点击当前标题按钮本身即可展开对应菜单。
- 图表结构菜单仍保留“编辑 / 移动 / 删除”现有动作集合，未修改图表数据模型、菜单状态机或 patch 提交逻辑。
- 图表数据表头的两处视觉问题已继续收口：列标题下方的多余留白已按块级布局修正，左上角“标签”角单元格也已补齐与同一行表头一致的高度和垂直对齐。
