# v0.2.11 更新记录

## Summary
- 编辑器主链路接入 `tiptap-markdown@0.8.10`，兼容当前 Tiptap 2.27.2。
- Markdown 加载、粘贴、复制和编辑器保存优先走 `tiptap-markdown`；PDF/export/diagnostics 侧暂保留原 `markdownCodec`。
- HTML 富文本粘贴不再由本地 sanitizer 或手动 `pasteHTML` 抢先接管，交回 Tiptap/ProseMirror 原生粘贴路径。

## Paste / Markdown
- 新增 `Markdown.configure({ html: true, transformCopiedText: true, transformPastedText: true })`。
- 移除编辑器粘贴入口中对复杂 HTML 的手动 sanitizer fallback，避免 Office/表格富文本被提前降级或触发二次插入错误。
- 新增 `tiptapMarkdownIntegration` 回归测试，覆盖 Markdown 标题/表格加载和 HTML 标题/表格默认 paste。

## Toast
- 长错误 toast 支持换行、断词和垂直滚动。
- `Position ... outside of fragment` 这类 ProseMirror 长错误改为短中文提示，避免 toast 内容横向溢出。
