# MDPad 更新说明

- 生成时间：2026-04-19 22:00（Asia/Shanghai）
- 文档类型：仓库内更新记录

## 更新概览

本轮更新文档聚焦近期已经落地的几项主要改进，便于后续查阅、整理发布说明或作为阶段性记录使用。

## 近期重点改进

### 1. HTML 预览编辑体验更稳定

- 修复了 HTML 行内编辑的退出交互。
- 点击外部区域时会先提交并退出编辑态。
- 双击切换到其他文本块时，会先提交当前内容，再进入新的编辑目标。
- 在打开链接、右键菜单或其他后续交互前，也会先收口当前编辑态，减少残留编辑框。

### 2. 页内目录锚点跳转不再导致空白页

- 修复了 HTML 预览中点击 `#hash` 锚点时可能触发错误导航的问题。
- 现在同页目录跳转会在当前预览区域内滚动到对应位置，不再把预览切成空白。
- 对中文标题、编码 hash 和缺失锚点场景也补充了对应处理与回归覆盖。

### 3. 右键导出菜单样式更加紧凑

- 收紧了菜单宽度、圆角、阴影和纵向留白。
- 菜单项的字体、间距和 hover 反馈都调整得更克制。
- 视觉方向更接近原生桌面应用，也与标题栏、状态栏风格更统一。

### 4. 版本发布整理已完成

- 近期已完成 `v0.2.5` 的发布整理。
- 发布说明重点覆盖外部文件同步安全、刷新恢复、Markdown 编辑稳定性，以及若干桌面端交互细节优化。

## 使用说明

- 后续更新文档统一保存在仓库根目录 `update/` 下。
- 文件名统一使用 `updatenote_yyyymmddhh.md` 规则，便于按时间排序和检索。
