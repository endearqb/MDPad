# 更新说明（2026-04-22 17:00）

## 本轮更新
- HTML 预览进入第一阶段可视化编辑：普通 HTML 元素现已支持在预览中选中、拖动与通过检查器修改常见样式，slide 类 HTML 也新增了 `Read / Present` 阅读与演示模式。
- 图表编辑链路完成了一轮集中收口：弹窗预览优先读取页面真实 runtime 图表快照，runtime-only 图表在无脚本复用时会回退到静态快照预览，首次保存也不再误报 binding changed。
- 图表数据弹窗已收缩为更纯粹的数据矩阵编辑器：展示设置与颜色编辑被移除，标签按列、系列按行进行排序，表头交互改为“拖拽手柄 + 输入框 + 条件删除”，整体更接近 Excel 风格。
- 旧 SVG 编辑能力已从 HTML 预览链路中完整下线，并补充了一份 SVG 编辑功能重写需求文档，为后续以更清晰边界重做该能力做准备。
