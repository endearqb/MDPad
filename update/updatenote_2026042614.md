# 修复 HTML 文本未改动退出误报

## Summary
- 修复 HTML 预览 inline 文本编辑中，未修改内容直接退出编辑框时误报 `Selected HTML text node could not be located.` 的问题。
- 保留真实文本修改的写回行为。

## Changes
- iframe host script 在 inline editor 退出时，如果 `nextText === originalText`，只恢复原文本节点，不再向父层发送 inline commit。
- `HtmlPreview` 父层收到未改动 inline commit 时直接忽略，避免触发源码 patch 和错误 banner。
- `applyHtmlTextPatch` 对 `currentText === nextText` 的 patch 先短路为 no-op success，不再依赖 locator 定位。

## Validation
- `pnpm exec vitest run src/features/editor/htmlPreviewDocument.test.ts src/features/editor/HtmlPreview.test.ts src/features/editor/htmlPreviewEdit.test.ts`
- `pnpm build`

# 移除 16:9 Slide 窗口模式

## Summary
- 下线窗口菜单中的 `16:9 Slide` 模式。
- 删除应用级 16:9 状态、窗口比例锁定、主内容区退出按钮和自定义 resize handles。

## Changes
- `TopBar` 窗口尺寸菜单只保留 `40% x 90%`、最大化和全屏。
- `App` 不再维护 16:9 模式状态，也不再监听 resize 后强制纠正窗口比例。
- `windowPreset` 删除 16:9 内容区窗口尺寸计算 helper。
- 清理 16:9 相关 i18n 文案、样式和测试。

## Validation
- `pnpm exec vitest run src/features/window/TopBar.test.ts src/App.test.ts src/shared/utils/windowPreset.test.ts`
- `pnpm build`
